Данный репозиторий содержит файл с проектом с автотестом REST API и jar файлом.
Что необходимо сделать перед запуском автотестов:
1. Запустить jar файл;
2. Войти на страницу по следующему адресу: http://localhost:8080/;
3. Нажать f12 для открытия devtools в браузере;
4. Перейти по вкладке "Application" -> "Сookies";
3. Скопировать значение конфигурационного параметра JSESSIONID;
(В данный момент  значение выглядит так: 
private static final String JSESSIONID = "679975347DF874F2F9F3D324C03C4A3D";)
Если вдруг автотест не запустится, попробуйте обновить страницу и снова 
поменять значение JSESSIONID.
